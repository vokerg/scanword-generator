# Scanword Generator MVP

Алгоритмический браузерный прототип генератора сканвордов под страницу A5.

## Что работает в версии 0.2

- процедурная генерация сканвордного шаблона;
- серые клетки с определениями и стрелками;
- горизонтальные и вертикальные ответы;
- до двух определений в одной клетке с диагональным разделителем;
- словарь из 500 выбираемых реальных русских слов;
- короткие определения для большинства слов и безопасный запасной текст;
- подбор слов по длине и уже известным буквам пересечений;
- несколько вариантов шаблона и перезапусков заполнения;
- режим показа ответов;
- экспорт A5 в SVG и проекта в JSON;
- полностью локальный запуск без сервера и зависимостей.

## Запуск

Можно открыть `index.html` напрямую в современном браузере.

Надёжнее запустить локальный сервер:

```bash
python3 -m http.server 8080
```

Затем открыть `http://localhost:8080`.

## Архитектура

1. `createMask` создаёт карту клеток вопросов.
2. `extractSlots` извлекает выходящие вправо и вниз последовательности клеток.
3. `fillSlots` подбирает ответы из словаря с учётом пересечений.
4. `buildFilledGrid` связывает определения, стрелки и ответы.
5. `renderSvg` собирает печатную страницу A5.

## Ограничения MVP

- шаблон пока эвристический и может оставлять неиспользованные белые клетки;
- направления ограничены вариантами вправо и вниз;
- перенос текста рассчитан на короткие определения;
- нет ручного редактора клеток;
- нет PDF-экспорта и отдельной страницы решений;
- словарь и определения пока встроены в JavaScript.
