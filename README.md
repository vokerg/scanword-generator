# Scanword Generator MVP

Первый алгоритмический прототип генератора сетки сканворда под страницу A5.

## Что уже работает

- детерминированная генерация пула из 500 псевдорусских слов по `seed`;
- размещение слов по горизонтали и вертикали;
- пересечения только по совпадающим буквам;
- клетки вопроса перед началом каждого ответа;
- несколько независимых попыток и выбор лучшей сетки;
- предпросмотр A5 в браузере;
- переключение показа ответов;
- экспорт SVG и JSON;
- запуск без сервера и без сторонних библиотек.

## Запуск

Откройте `index.html` в современном браузере.

Для локального HTTP-сервера можно выполнить:

```bash
python3 -m http.server 8080
```

После этого откройте `http://localhost:8080`.

## Ограничения MVP

Это пока тест геометрии и алгоритма раскладки, а не полноценный издательский генератор:

- слова синтетические и не имеют настоящих определений;
- клетка вопроса содержит только номер и направление;
- нет редактора сетки;
- нет обязательных и приоритетных слов;
- нет разбиения длинного определения внутри клетки;
- нет PDF-экспорта — SVG уже имеет физический размер A5 и печатается без потери качества.

## Следующий технический этап

1. Формат входного словаря `answer + clue + priority`.
2. Обязательные слова и функция оценки покрытия.
3. Полноценные клетки определений и несколько стрелок из одной клетки.
4. Локальный редактор сетки.
5. Экспорт страницы с ответами и печатного PDF.
