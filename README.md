# Scanword Generator MVP

Браузерный прототип алгоритмического генератора сетки для сканворда формата A5.

## Что умеет

- содержит встроенный словарь из 500 настоящих русских слов;
- выбирает из словаря пул в 500 слов по заданному seed;
- строит до 18 вариантов сетки и выбирает лучший;
- размещает слова горизонтально и вертикально с проверкой пересечений и соседних клеток;
- показывает предпросмотр A5, статистику и список размещённых слов;
- экспортирует SVG и JSON;
- работает локально без сервера приложений и внешних зависимостей.

Определения пока не встроены: в клетках вопросов отображаются номер и стрелка. Этот MVP проверяет геометрию и эвристику раскладки.

## Запуск

```bash
python3 -m http.server 8080
```

Откройте `http://localhost:8080`.
