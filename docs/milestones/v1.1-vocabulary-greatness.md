# Milestone 1.1 — Truthful vocabulary and editorial corpus expansion

Date: 2026-07-18

Status: accepted draft-project baseline.

## Decision

Squash-merge the vocabulary 1.1 research line into `main`.

The accepted baseline adds:

- a 40,966-entry attributed corpus v8;
- explicit distinction between generic clues and sourced factual templates;
- preferred Russian GeoNames and collision-safe fallback handling;
- 1,469 filtered non-city geographic entities;
- browser-equivalent Node benchmark bootstrapping;
- an opt-in adaptive portfolio mode;
- reproducible negative-result builders and research documentation.

The project is itself a draft, so integration is not blocked on a 50-seed promotion ceremony. Unfinished validation remains explicit debt and moves to the next research branch.

## Runtime path

The default generator architecture remains:

```text
40,966-entry attributed source corpus
-> deterministic active sets at 2,500 and 3,500 entries
-> complete construction and clue allocation
-> same-geometry editorial repair
-> panel-first candidate selection
-> complete validation
```

Adaptive early acceptance is implemented but not the browser default.

## Canonical-name policy

The selected builder:

1. uses language-tagged preferred Russian GeoNames;
2. drops a fallback alias when its preferred replacement collides with an admitted answer;
3. uses filtered surface clusters only as fallback evidence;
4. permits a small source-ID keyed letters-only exception map;
5. validates representative canonical answers during generation.

## Accepted debt

This milestone intentionally accepts:

- no completed 50-seed density checkpoint on the committed 40,966-entry chunks yet;
- full portfolio runtime around 20–25 seconds;
- generic proper-name clues still present;
- factual templates that are sourced but not publication-edited prose;
- a geographically weighted entity expansion before arts, science, sport and history receive equivalent treatment;
- adaptive mode remaining experimental.

It does not claim zero-panel construction or publication-ready clues.
