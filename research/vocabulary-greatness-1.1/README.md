# Vocabulary Greatness 1.1

Status: active research branch.

## Objective

Improve vocabulary-first 1.0 without losing its density frontier.

The work is split into independently measurable tracks:

1. reduce generic template-clue debt;
2. broaden attributed entity coverage beyond cities and personal names;
3. improve category diversity inside seed-specific active sets;
4. reduce the cost of the 2,500/3,500 portfolio through shared preparation;
5. validate all changes on identical seeds with complete structural checks.

## Promotion boundary

Nothing in this directory changes production status by itself. A candidate may be promoted only after:

- corpus audit passes with zero invalid and zero duplicate normalized answers;
- browser defaults and load order are explicit;
- complete grid validation passes;
- identical-seed metrics include per-seed regressions and runtime;
- negative results and accepted debt are documented;
- a 50-seed checkpoint confirms the selected direction.

## Baseline

Vocabulary-first 1.0:

- 39,586 generated attributed entries;
- deterministic 2,500/3,500 active-set portfolio;
- same-geometry editorial repair;
- 20-seed release result: 5.30 residual panels, 48.45 answers, 53.00 crossings;
- approximate release-gate runtime: 21–25 seconds;
- template clues: 15,136 / 38.24%.
